Loaderlet: A Simple Javascript Loader

Project Page

Test Page: Open your console and call function lodrlt: lodrlt();

You can drop this link Loaderlet to the bookmark bar, to push-load famous javascript libraries. such as:

  1. jQuery
  2. Zepto
  3. Backbone
  4. Underscore
  5. Mordernizr
  6. D3
  7. Raphael
  8. EaselJS
